# BMI Calculator Android App

This is a simple BMI Calculator app built with Jetpack Compose.

## Features
- **Calculate BMI**: Enter your weight and height to get your BMI score.
- **Unit Systems**: Support for both Metric (kg/cm) and Imperial (lbs/ft+in) units.
- **Categorization**: Visual feedback for Underweight, Normal, Overweight, and Obese categories.
- **Ideal Weight**: Calculates the ideal weight range for your height.
- **Persistence**: Remembers your last selected unit system.

## How to Run
1. Open the project in Android Studio.
2. Sync Gradle files.
3. Run the app on an emulator or physical device.

## Technologies
- Kotlin
- Jetpack Compose
- Material Design 3
